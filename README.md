# Fernand Gbodjo — Personal Portfolio

CSIT 212 Final Project. A single-page portfolio site built with React, Vite, and Tailwind CSS, deployed on Netlify.

## Sections

- **Hero** — name, tagline, animated gradient background
- **About Me** — bio, location, interests
- **Projects** — four shipped healthcare apps (HIPAA Shield, MedSort AI, ShiftRx, CareAlert) with live demos and source links
- **Skills** — tools and technologies
- **Sikavia** — DTC e-commerce brand
- **How I Built This** — prompts used to build the site (assignment requirement)
- **Contact** — email and GitHub

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The static site is output to `dist/` and deployed to Netlify.

## Stack

React · Vite · Tailwind CSS · Lucide Icons · Netlify
